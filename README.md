# いろんなチャンピオンをぱっと使ってみた感想

## Twistedfate

### ざっくりスキル解説

- Q: 貫通する優秀な方向指定攻撃。ハラス。
- W: 通常攻撃強化
- E: 通常攻撃強化
- R: 移動スキル

### ざっくり解説

 - 基本ロールはMid
 - 攻撃技はAAとQのみなので、AAハラスに気をつける（得にLv3以降）
   - WとEでAAが強化される
   - つまりAAで継続的にダメージを出すほうが得意
   - Wはマナ回復の効果もあるのでレーン居残り能力は高め？
 - 1v1してる時にUltでのGankはキツイ
   - 頭の上に丸いやつ出てきたら注意
 - そのかわり集団戦でUltの有無は気にしなくて良い
